What is TRIVIA API

Trivia API is a QA testing studies final project, using different programming languages to let you access to API Open Trivia Database. This API is free to use and meant for projects for students like us.

FAQ

How it works?

To access to the trivia questions you can from the home page or directly from our header. You just need to click Trivia

What are the categories?

In our API, you can choose between 24 categories, including movies, history and nature! Just choose one of these on the list and play!

Which are the difficulty levels?

You can choose between three different levels of difficulty: easy, medium and hard. The questions will be curated according to these difficulties, and if you don't choose any, you will receive from all difficulties.

The question doesn't appear, what is the problem?

We are dependent on an external database to get our questions and answers. If this page is down, its impossible for the API to access these. You can access the status of OpenTDB in this page

Can I access to the Github page?

In our page you can see our code and the inner workings of the page, along with the technical and user manuals.
